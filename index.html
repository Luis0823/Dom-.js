Dom++.js

Dom++.js es una pequeña biblioteca js para la manipulación DOM

¿Por qué usar Dom++?

Con Dom++ puedes:
  • Seleccionar elementos velozmente
  • Agregar estilo a elementos de forma potente
  • Crear y agregar elementos más rápido
  • Crear bucles avanzados y tiempos de espera
  • Administrar atributos de clases y estilo css
  • Es mucho mas pequeño y compacto que otras bibliotecas

Seleccionar elementos

Generalmente en js regulares utlizamos los metodos document.getElementById() para seleccionar un elemento usando su id, document.getElementByClassName() para seleccionar usando clases y querySelector() para cualquiera de los dos.

Pero en Dom++ utilizamos esta sintaxis para seleccionar un elemento

Elemento Dom++ new X("#div");
Elemento DOM normal $(#div);
Esto 👆 selecciona un elemento con "div" como Id(Identificador)

Elemento Dom++ new X(".div");
Elemento DOM normal $(.div);
Esto 👆 selecciona elementos con "div" como Class(Clase)
Elemento Dom++ new X("div");
Elemento DOM normal $(div);
Esto👆 selecciona todas las etiquetas <div></div>

Los elementos seleccionados también se pueden pasar a una variable.

Elemento Dom++ var div = new X("#div");
Elemento DOM normal var div = $(#div);

Métodos

El objeto X y $ tienen muchos métodos útiles como ocultar y mostrar

Ocultar y mostrar

Un elemento se puede ocultar usando método hide()
Elemento Dom++ var div = new X("#div");
div.hide();
Elemento DOM normal var div = $("#div");
hide(div);
El código anterior selecciona un elemento y lo oculta, para mostrar un elemento oculto utiliza show() ↓
Elemento Dom++ var div = new X("#div");
div.show();
Elemento DOM normal var div = $("#div");
show(div);

Cambiar HTML

El HTML de un elemento se puede cambiar usando el método html()
Elemento Dom++ var div = new X("#div");
div.html("<h2>Test</h2>");
Elemento DOM normal var div = $("#div");
html(div, "<h2>Test</h2>");

Cambiar el texto

El texto de un elemento se puede cambiar usando el método txt()
Elemento Dom++ var div = new X("#div");
div.txt("Test");
Elemento DOM normal var div = $("#div");
txt(div, "Test");

Cambiar estilo

El estilo CSS de un elemento se puede cambiar usando el método de style()
Elemento Dom++ var div = new X("#div");
div.style("display", "block");
Elemento DOM normal var div = $("#div");
style(div, "display", "block");
style() toma la propiedad de 2 parámetros al estilo y el valor que se asignará a la propiedad especificada
Para obtener un valor de propiedades CSS especificado, simplemente deja el parámetro de propiedad en blanco
Elemento Dom++ var div = new X("#div");
div.style("display");
Elemento DOM normal var div = $("#div");
style(div, "display");

Cambiar atributos

Para cambiar un atributo usa el método attr()
Elemento Dom++ var div = new X("#div");
div.attr("class", "div");
Elemento DOM normal var div = $("#div");
attr(div, "class", "div");
También puedes obtener el valor de un atributo dejando el parámetro de valor en blanco ↓
Elemento Dom++ var div = new X("#div");
div.attr("class");
Elemento DOM normal var div = $("#div");
attr(div, "class");

Eliminar atributos

El atributo de un elemento puede eliminarse mediante el método reAttr
Elemento Dom++ var div = new X("#div");
div.reAttr("class");
Elemento DOM normal var div = $("#div");
reAttr(div, "class");

Agregar y quitar clases

las clases se pueden agregar usando el método addClass() y las clases se pueden eliminar usando el método reClass
Elemento Dom++ var div = new X("#div");
div.addClass("div");
div.reClass("div");
Elemento DOM normal var div = $("#div");
addClass(div, "div");
reClass(div, "div");

Obtener valor

Para obtener y establecer el valor de las entradas y las áreas de texto, puedes usar el método val() ↓
Elemento Dom++ var input = new X("#div");
input.val("Test");
Elemento DOM normal var input = $("#input");
val(input, "Test");
Para obtener el valor simplemente deja el parámetro de valor en blanco
Elemento Dom++ var input = new X("#input");
var value = input.val();
Elemento DOM normal var input = $("#input");
var value = val(input);

Buscar eventos

Para verificar si hay un evento, usa el método checkFor(), es un reemplazo para addEventListener
Elemento Dom++ var div = new X("#div");
div.checkFor("click","console.log('Test')");
Elemento DOM normal var div = $("#div");
checkFor(div, "click","console.log('Test')")
#Nota: El ultimo parámetro es el código que se ejecutará si el evento se realiza, puede ser una cadena o una plantilla literal

Método box()

El método box() te permite establecer el ancho, alto y color de fondo del elemento seleccionado
Elemento Dom++ var div = new X("#div");
div.box("50px","50px","dark");
Elemento DOM normal var div = $("#div");
box(div, "50px","50px","dark");

Funciones

Crear elementos

Crear elementos y agregarlos es una molestia en js regulares, pero en Dom++ es fácil con la funcion create()
Elemento Dom++ var div = new X("#div");
create("p","#p", div);
Elemento DOM normal var div = $("#div");
create("p", null, div);
El elemento se agrega automaticamente a la etiqueta <body></body> si el ultimo parametro esta vacio El segundo parámetro se usa para establecer una clase o identificador del elemento recién creado para utlizarlo mas tarde
Utilizar el elemento creado
El elemento recién creado puede pasarse a una variable que puede usar todos los métodos de Dom++
Elemento Dom++ var div create("p","#p", div);
div.txt("Test");
Elemento DOM normal var div = create("p", null, div);
txt(div, "Test");

Creando clases

Las clases se pueden crear en js usando la función de Dom++ makeClass()
makeClass("div","display:block;");
El primer parámetro es el nombre de la clase y el segundo es el CSS dentro de la clase

Temporizador

Utiliza la funcion timer() para ejecutar una función después de una cantidad específica de tiempo
Elemento Dom++ y DOM normal timer("console.log('Test');", 2);
El temporizador toma dos parámetros, el primer parámetro toma el código que debe ejecutarse y el segundo parámetro es el tiempo en segundos(1 Segundo = 1000 Milisegundos)

Repetidor

La función de repetición se utiliza para repetir la ejecución de un código varias veces
repeat("console.log('Test')",5);
El primer parámetro es el código a ejecutar y el segundo es el número de veces que se repite
Repitiendo para siempre
La función de repetición también tiene un tercer parámetro que se usa para que se repita para siempre
repeat("console.log('Test')",2,-1);
Escribe -1 ya que el tercer parámetro lo repite para siempre
Intervalo de repatacion
El intervalo de repetición es el cuarto parámetro de la función de repetición que se utiliza para establecer un intervalo de tiempo entre cada nueva repetición
repeat("console.log('Test')",2,1,5)

Encriptacion y Hash

Encriptador y desencriptador

Utiliza los diferentes encriptadores y desencriptadores de Dom++
Elemento Dom++ y DOM normal encodeBinary("Test"); decodeBinary("1010100 1100101 1110011 1110100 "); encodeUTF8("Test"); decodeUTF8("Test"); encodeBase64("Test"); decodeBase64("VGVzdA=="); encodeSwich("Test"); decodeSwich("ImUMvmpczaMII01DFFwMEt=="); cc("Test"); cc("Gvhg");
El encriptador cc() encripta y desencrpta el texto de la misma manera ya que esta basado en Caesars Cipher

Hash

La funcion md5() de Dom++ crea un hash de la misma forma que lo hace en el lenguaje php
Elemento Dom++ y DOM normal md5("Test");
